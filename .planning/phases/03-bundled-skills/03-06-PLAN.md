agentskills.io spec requirements for external skills:
  - Folder name: skills/external/<lowercase-hyphen-name>/
  - SKILL.md first line (H1): # [Skill Name] (via [source-org]/[source-repo])

  - SKILL.md body: under 500 lines

  - License: must be compatible with this repo's license (check source repo LICENSE file)
