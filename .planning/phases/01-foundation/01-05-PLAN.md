    Create internal/service/uninstall.go:
    - Define local interfaces: uninstallAdapter (RemoveMCPConfig, RemoveSkill), uninstallStore (GetRecord, RemoveRecord)
    - Sentinel error: ErrNotInstalled
    - Type UninstallService struct with adapter uninstallAdapter, store uninstallStore
    - Constructor NewUninstallService(ad adapter.AssistantAdapter, store *config.ConfigStore) *UninstallService
    - Method Uninstall(name string) error:
      1. rec, found, err = store.GetRecord(name) — if !found return ErrNotInstalled
      2. err = adapter.RemoveMCPConfig(name) — on error return err (do not proceed to RemoveRecord)
      3. If rec.Type == domain.PackageTypeSkill: err = adapter.RemoveSkill(name) — on error return err
      4. err = store.RemoveRecord(name) — return err

    Create internal/service/update.go:
    - Define local interfaces: updateRegistry (Resolve), updateStore (GetRecord, ListInstalled), updateInstaller (Install)
    - Type UpdateService struct with registry updateRegistry, store updateStore, installer updateInstaller
    - Constructor NewUpdateService(reg *registry.RegistryManager, store *config.ConfigStore,
      installSvc *service.InstallService) *UpdateService
      Note: InstallService injection here means D-08 ownership is automatically satisfied because
      InstallService calls WriteMCPConfig which checks installed.json ownership.
    - Method Update(name, target string) (string, error):
      1. rec, found, err = store.GetRecord(name) — if !found return "", ErrNotInstalled
      2. latest, err = registry.Resolve(name) — on error return "", err
      3. If latest.Version == rec.Version: return "already up to date", nil
      4. _, err = installer.Install(name, target) — return "", err on failure
      5. Return fmt.Sprintf("updated %s: %s → %s", name, rec.Version, latest.Version), nil
    - Method UpdateAll(target string) ([]string, error):
      1. records, err = store.ListInstalled() — on error return nil, err
      2. For each record: call Update(record.Name, target), collect results and errors
      3. Return all result messages and first non-nil error (continue updating others even on single error)

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
