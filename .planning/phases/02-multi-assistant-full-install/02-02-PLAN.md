## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| manifest → adapter | MCPServerEntry (command, args, env) from registry flows into JSON written to ~/.copilot/mcp-config.json or VS Code mcp.json |
| $COPILOT_HOME env var | User-controlled env var influences where CopilotCLIAdapter writes; must be joined safely |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-04 | Tampering | CopilotCLIAdapter mcpConfigPath — $COPILOT_HOME | mitigate | filepath.Join(env, "mcp-config.json") — env value treated as directory path only; no shell execution |
| T-02-05 | Tampering | WriteMCPConfig foreign key clobber | mitigate | map[string]interface{} merge preserves all unmanaged keys; conflict check returns ErrForeignConflict for non-owned keys |
| T-02-06 | Information Disclosure | VS Code edition detection leaks config dir structure | accept | Detection is read-only stat(); no sensitive data returned; failure defaults to "Code" path safely |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | No new dependencies in this plan |
</threat_model>
