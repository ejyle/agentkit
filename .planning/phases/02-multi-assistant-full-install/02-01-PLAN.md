## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | UvxInstaller.Install / DockerInstaller.Install | mitigate | exec.Command arg-array form only — never exec.Command("sh", "-c", combined string). Package name from manifest is passed as a separate arg element, never interpolated into a shell string. |
| T-02-02 | Spoofing | DockerInstaller docker pull | mitigate | spec.Package = image:tag from registry manifest which is SHA256-verified in Phase 1 registry fetch. Agentkit does not compose image names from user input. |
| T-02-03 | Tampering | SkillInstallPath path construction | mitigate | filepath.Join() used for all path construction; target string validated against allowlist in cmd/root.go before reaching SkillInstallPath; no user-controlled path segments injected. |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | Phase 2 adds no new Go module dependencies per Package Legitimacy Audit in RESEARCH.md. No new slopcheck required. |
</threat_model>
